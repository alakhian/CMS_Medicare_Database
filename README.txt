Directory descriptions:
HCPCS_Filter/ contains a Java script for creating a filter for HCPCS codes.
RMarkdown/ contains the final report.
sql/ contains relevant SQL queries and DDL scripts that were used.
Tableau/ contains the Tableau workbooks with the visualizations that were used.
workflows/ contains the .xml files for the Oracle Data Miner workflows
Physician_anomalies_sorted.txt contains the actual text from the 
	Physician_Anomaly database of only the predicted anomalous data points.
